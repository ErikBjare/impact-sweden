Cheatsheet for Python
=====================

Add two numbers together and print the result:

```python
print(1+2)
```

Calculate the hypotenuse of a right-angled triangle:

```python
base = 4  # The base
side = 2  # The side
hypotenuse = sqrt(base**2 + side**2)
print(hypotenuse)
```

Print the power 2 of the first 10 whole numbers:

```python
for i in range(10):
    print(i**2)
```
